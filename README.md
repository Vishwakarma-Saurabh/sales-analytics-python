📊 Sales Analytics Project

A data analytics pipeline built using Python, SQLite, Pandas, Matplotlib, and Seaborn to analyze sales data and generate business insights through visualizations.

This project demonstrates how a real-world data analyst workflow works — from data storage to insight generation.

🚀 Project Overview

Businesses generate large amounts of transactional sales data.
This project shows how to:

Store structured data using SQLite

Process and analyze data with Pandas

Generate insights using Matplotlib & Seaborn

Build a modular analytics system using Python packages

🛠️ Tech Stack
Technology	Role
Python	Core programming language
SQLite	Data storage
Pandas	Data processing
Matplotlib	Data visualization
Seaborn	Statistical visualization

⚙️ How to Run the Project

1️⃣ Clone the repository
git clone https://github.com/Vishwakarma-Saurabh/sales-analytics-project.git
cd sales-analytics-project
2️⃣ Install dependencies
pip install -r requirements.txt
3️⃣ Run the project
python run.py

The program will:

Load sales data from SQLite

Perform analysis using Pandas

Generate visual reports
